# adote-um-aumigo
Avaliação 2 Adote Um Aumigo

🐶 Adote um Amigo

Objetivo:

Desenvolver um site institucional para uma ONG de adoção de cães utilizando HTML, CSS e JavaScript, aplicando os conceitos estudados em sala de aula como estruturação de páginas, manipulação do DOM, eventos e validação de formulários.

Tecnologias Utilizadas:

HTML

CSS

JavaScript


Estrutura das Páginas/Seções:
Home

Sobre

Processo de Adoção

Nossos Amigos

Histórias de Sucesso

FAQ

Contato


Funcionalidades Implementadas:

Menu de navegação

Rolagem suave entre seções

Menu responsivo para dispositivos móveis

Modo escuro (Dark Mode)

Contadores animados

FAQ interativo

Animações ao rolar a página

Formulário com validação de campos

Mensagem de confirmação de envio
